# Python for Data Analysis — 8-Week Study Plan

## Week 1: Python Fundamentals
### Learning Objectives
- Understand variables, data types, and operators
- Write and run your first Python scripts
- Use lists, tuples, and dictionaries confidently

### Resources
- Python.org official tutorial (free)
- Corey Schafer's Python Beginner series on YouTube

### Practice Exercise
Build a script that reads a list of numbers and prints the mean, min, and max.

### Checkpoint
What is the difference between a list and a tuple in Python?

---

## Week 2: Control Flow and Functions
### Learning Objectives
- Write if/else statements and for/while loops
- Define and call reusable functions
- Understand scope and return values

### Resources
- Automate the Boring Stuff with Python (free online — chapters 1-3)
- Real Python — Functions tutorial

### Practice Exercise
Write a function that takes a list of student scores and returns a grade (A/B/C/D/F).

### Checkpoint
What does the return keyword do and what happens if you omit it?

## Key Milestones
- Week 2: 25% — Python syntax confident
- Week 4: 50% — Pandas and data wrangling working
- Week 6: 75% — First end-to-end analysis complete
- Week 8: Completion — Portfolio project on GitHub
